Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head element contains metadata that will help the browser render the website.
  • The body element is an example of a semantic HTML element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You have to link your CSS styling in your HTML file in the head after meta and before title.
  • A CSS rule will contain a selector that defines the element or attributes to which the declaration will apply.
  • A declaration contains a property followed by the value for that property.
